# Job Portal

A full-stack Job Portal web application that connects job seekers with recruiters. Users can browse and apply for jobs, while recruiters can post and manage job listings.

## Features

- User Authentication & Authorization
- Job Search and Filtering
- Apply for Jobs
- Recruiter Dashboard
- Post, Update, and Delete Jobs
- Manage Applications
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Structure

```
Job-Portal/

├── client/      # Frontend
├── server/      # Backend
└── package.json
```

## Installation

### Clone the repository

```bash
git clone https://github.com/prakhar062004/Job-Portal.git
cd Job-Portal
```

### Install dependencies

```bash
npm install
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```

## Run the Project

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```
